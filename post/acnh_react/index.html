Animal Crossing PWA App

July 2020 One-minute read

A Progressive Web App (PWA) build on React and Material UI which filters Animal Crossing information.

Duration: 1 week passion project

Context

I created this app to demonstrate my knowlege of new technologies I learned at my internship, and because I love Animal Crossing: New Horizons! Technologies include: Progressive Web Apps (PWA), React, Material UI, and response design.
